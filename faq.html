Frequently Asked Questions

Find answers to common questions about our tools and services

Are my files uploaded to your server when I use your tools?

No, your files are never uploaded to our servers. All processing happens directly in your browser using JavaScript. Your files never leave your device, ensuring complete privacy and security.

Can I use PicsPanda offline?

Yes! Our application is a Progressive Web App (PWA), which means you can install it on your device and use it even without an internet connection. The tools will work just as they do online.

Are there file size limits for uploads?

Our tools are designed to handle files up to 50MB in size. This limit helps us ensure good performance for all users while still accommodating most common use cases.

How do I split a PDF into multiple files?

Use our PDF Split tool. Upload your PDF, enter the page ranges you want to extract (like '1-3, 5, 7-9'), and click 'Split PDF'. You'll receive separate PDF files for each specified range.

Can I convert multiple images at once?

Currently, our tools process one file at a time for optimal quality control. We're working on batch processing features for future updates.

What's the difference between lossy and lossless compression?

Lossy compression permanently removes some image data to reduce file size, resulting in smaller files but potential quality reduction. Lossless compression preserves all original data and quality, but achieves less dramatic file size reduction.

Do I need to create an account to use these tools?

No, all our tools are completely free to use without any account registration. Just visit our website and start using the tools immediately.

What image formats can I convert between?

Our converter supports the following formats:

  • JPEG/JPG
  • PNG
  • WebP
  • GIF
  • TIFF
  • BMP

You can convert between any of these formats using our image converter tool.

How can I report a bug or suggest a new feature?

We welcome your feedback! Please use our contact form to report bugs or suggest new features. Be sure to include as much detail as possible to help us understand your issue or idea.

Is PicsPanda really free? What's the catch?

Yes, PicsPanda is completely free! There's no catch. We believe in providing accessible tools for everyone without barriers. We may add premium features in the future, but our core tools will always remain free.
